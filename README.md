Sales Analysis Dashboard | Power BI

Project Overview
This project is an interactive Power BI Sales Analysis Dashboard designed to monitor sales performance across different countries, products, and salespersons. The dashboard provides key business insights through visualizations, KPI cards, and interactive filters.

Features
* Interactive slicers for Country, Product, and Salesperson
* KPI Cards for Total Sales, Boxes Shipped, Countries, and Products
* Sales Trend Analysis over time
* Top Products Analysis
* Sales by Country Analysis
* Top Salespersons Analysis
* Country Contribution Percentage
* Reset Filters Button using Bookmarks
* Dynamic Key Insights Section

Key Metrics
* Total Sales: 6M
* Total Boxes Shipped: 177K
* Total Countries: 6
* Total Products: 22

Key Insights
* Australia generated the highest sales among all countries.
* Smooth Silky Salty was the top-selling product.
* Devrat Damarsingh achieved the highest sales among salespersons.
* January recorded the highest sales performance.

Tools & Technologies
* Power BI
* DAX
* Data Visualization
* Data Analysis
* Bookmarks & Buttons
* Interactive Slicers
